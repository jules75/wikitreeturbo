WikiTreeTurbo

What does WikiTreeTurbo do?

It provides these extra features for the genealogy web site WikiTree.com;

  • Quick search with backquote key (`)
  • Tag faces in photos
  • Collapsible descendant lists

Can I get help or suggest a feature?

Yes please, email the developer at julianlaffey@gmail.com

How do I add people to an already tagged photo?

You have to tag everyone again.

Can I edit or delete the funny looking code comments WikiTreeTurbo leaves on images?

These comments make image tagging work. It's ok to delete them, but you'll lose your image tags.

If you edit the code comments, image tags might break.

It's ok to have multiple code comments - WikiTreeTurbo only uses the newest one.
